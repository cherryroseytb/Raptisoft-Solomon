# 게임 설명 추가 메모

## Boneyard

### 1) 기본 진행
- 정해진 맵에서 몬스터를 처치하며 진행.
- 기본 맵 1개가 있고, 기능 해금으로 골드를 써서 추가 맵 3개 오픈 가능.
- 플레이 중 획득 가능 요소:
- 골드
- 랜덤 스킬 보너스(파란색)
- 레벨업과 동일한 3지선다 스킬 보너스(빨간색)
- 일정 시간 데미지 4배 보너스
- 반지 아이템(매직/레어/에픽 등급, 에픽 이상은 희귀)

### 2) 골드/반지/인벤토리
- 골드는 적 처치 시 랜덤 드롭.
- 반지도 종종 드롭됨.
- 반지는 인벤토리에서 최대 2개 착용 가능.
- 반지 선택 시 `착용` 또는 `분해` 가능.
- 분해 시 해당 단판(run) 한정 랜덤 버프 획득.

### 3) 사망 후 반지 계승(무덤 시스템)
- 반지를 인벤토리에 가진 상태로 사망하면, 마을 NPC(무덤지기)를 통해 골드를 지불해 반지 계승 가능.
- 비용은 랜덤(보통 100~400 골드).
- 지금까지 사망하면서 나온 반지들 중 최대 2개를 골라 다음 판으로 가져갈 수 있음.
- 가져간 반지는 게임 내에서 분해하지 않는 한, 다음 사망 시 다시 무덤으로 돌아가 재선택 가능.
- 반지를 반복 선택하다 보면 무덤지기 화면에서 반지가 떨리는(진동하는) 상태가 발생할 수 있음.
- 떨림 상태의 반지는 해당 판 사용 후 확률적으로 파괴될 가능성이 있는 것으로 보임.
- 관측 메모: 떨림 상태에서 3회 사용 사례는 있었으나 실제 파괴는 미확인.

### 4) 마을 NPC 및 영구 해금
- 시작 직후에는 안 보이고, 스테이지를 클리어하면서 마을 기능이 순차적으로 열림.
- 메인 NPC 오른쪽:
- 골드를 지불해 스킬/기능을 영구 구매 가능.
- 예시: `시작 시 반지 2개 랜덤 지급`, `시작 시 기초 스킬 +2`.
- 구매한 스킬/기능은 슬롯에 장착해야 다음 판에 적용됨.
- 슬롯 기본 2칸, 1000골드로 임시 확장 가능(최대 칸수 제한 존재).
- Boneyard 기준으로 `판 종료 = 사망 = 마을 복귀`이며, 이 시점에 슬롯 임시 확장은 종료됨.
- 메인 NPC 왼쪽:
- 마법사(무덤지기) 등장.
- 랜덤 골드 지불 후, 누적된 사망 반지 풀에서 최대 2개를 선택해 다음 판 반입 가능.

### 5) 무덤 반지 풀 포함 범위(확정)
- 사망 순간 기준, 게임 내 인벤토리에 있는 반지가 풀에 포함됨.
- 착용/미착용 모두 포함.

### 6) 추가 확인 필요
- 반지 파괴의 정확한 트리거(선택 횟수 임계값, 파괴 확률, 예외 조건).

### 7) 레퍼런스 이미지 기반 확인 사항
- 경로: `Solomon's Boneyard/copygame/ref_images`
- 로비에서 상호작용하는 NPC/메뉴가 분리됨:
- 메인 NPC 메뉴
- 무덤지기 메뉴(반지 선택 페이지 포함)
- 상인/기능 메뉴
- 캐릭터 선택/인벤토리 메뉴
- 해금 메뉴가 `Services` / `Fates`로 분리되어 보임.
- 초기 4캐릭터 외에 다수 캐릭터 해금 상태 확인됨.
